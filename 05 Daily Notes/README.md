# Daily Notes

Dein persönliches Logbuch. Täglich Gedanken, Erlebnisse und Reflexionen dokumentieren.

## Format

Dateiname: `YYYY-MM-DD.md` (z.B. 2026-04-27.md)

Obsidian kann automatisch Daily Notes erstellen. Command Palette: "Create today's note"

## Struktur einer Daily Note

- **Erledigt** — Was hast du heute geschafft?
- **Nächste Schritte** — Was kommt als nächstes?
- **Gelernt** — Neue Erkenntnisse?
- **Verknüpfungen** — Links zu Projekten, Bereichen, Ressourcen

## Wöchentliche Review (Sonntags)

Nutze die Notizen der Woche für eine kurze Reflexion:
- Welche Muster siehst du?
- Welche Erkenntnisse sind wertvoll?
- Was in Projekte oder Bereiche übertragen?

## Monatliche Review (Monatsende)

- Gesamtzusammenfassung des Monats
- Fortschritt in Projekten und Areas prüfen
- Neue Erkenntnisse dokumentieren

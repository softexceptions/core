# Claude Code — Trigger-Phrasen

| Phrase | Was passiert |
|---|---|
| **„Was war aktuell?"** | Kontext-Briefing: letzte Daily Notes + aktive Projekte lesen |
| **„Session-Ende"** | Daily Note erstellen, Erkenntnisse speichern, Inbox aufräumen |
| **„Lege die Daily Note an"** | Nur Daily Note erstellen |
| **„Merke dir den Plan"** | Aktuellen Claude-Plan als Notiz in den Vault speichern |
| **„Merk dir das"** | Inhalt in passende Kontext-Datei in `00 Kontext/` speichern |
| **„Auto-Accept"** | Claude akzeptiert alle Aktionen ohne Rückfrage (Standard-Modus aktivieren) |

## Konfiguration

| Datei | Zweck |
|---|---|
| `permissions.json` | Definiert, was Claude ausführen darf und was nicht |

# core

Diese Datei richtet dir automatisch ein personalisiertes Vault in Obsidian ein.
Du beantwortest ein paar Fragen — Claude baut den Rest.

---

## Was du brauchst

- [Obsidian](https://obsidian.md) — kostenlos herunterladen
- Claude Cowork oder Claude Code — auf [claude.ai](https://claude.ai)

---

## So funktioniert es

1. Obsidian öffnen und einen neuen Vault erstellen
2. `CLAUDE.md` in den Vault-Ordner legen
3. Claude Cowork starten und den Vault-Ordner als Kontext hinzufügen
4. Claude startet automatisch das Setup und führt dich durch alles

Am Ende hast du eine fertige Ordnerstruktur, persönliche Kontext-Dateien und eine CLAUDE.md die bei jeder Session automatisch geladen wird.

---

## Was danach passiert

- Claude kennt deinen Namen, deinen Stil und deine Projekte
- Jeder Chat startet mit vollem Kontext — kein Wiederholen mehr
- Sag "merk dir das" und Claude speichert es an der richtigen Stelle
- Das System wird mit jeder Session besser
